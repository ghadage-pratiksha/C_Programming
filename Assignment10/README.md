# C Programming - Assignment 10

## 📖 Description

This repository contains C language solutions for Assignment 10. The programs focus on mathematical calculations and unit conversions using functions. The assignment covers area calculations, distance conversion, temperature conversion, and square unit conversion to strengthen programming fundamentals and logical thinking.

## ✨ Features

- Function-Based Programs
- User Input Handling
- Mathematical Calculations
- Unit Conversions
- Area Calculations
- Floating-Point Operations
- Time Complexity Analysis

## 📂 Programs

| Question | Description |
|----------|-------------|
| Q1 | Calculate the area of a circle |
| Q2 | Calculate the area of a rectangle |
| Q3 | Convert kilometres to meters |
| Q4 | Convert Fahrenheit to Celsius |
| Q5 | Convert square feet to square meters |

## 🛠 Technology

- C

## ▶️ Running the Programs

Compile the program:

```bash
gcc filename.c -o output
```

Run:

```bash
./output
```

Windows:

```bash
output.exe
```

Example:

```bash
gcc Q1_CircleArea.c -o Q1
Q1.exe
```
