# C Programming - Assignment 3

## 📖 Description

This repository contains C language solutions for Assignment 3.

The programs are written to practice basic programming concepts including:

- Functions
- Loops
- Conditional Statements
- Factors of a Number
- Even Numbers
- Character Handling
- Case Conversion
- Vowel Checking

## 📂 Programs

| Question | Program |
|----------|---------|
| Q1 | Print first N even numbers |
| Q2 | Display all factors of a number |
| Q3 | Display even factors of a number |
| Q4 | Convert uppercase to lowercase and lowercase to uppercase |
| Q5 | Check whether a character is a vowel |

## 🛠 Language

- C

## ▶️ How to Run

Compile the program:

```bash
gcc filename.c -o output
```

Run:

```bash
./output
```

or on Windows:

```bash
output.exe
```
