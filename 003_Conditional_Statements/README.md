# Conditional Statements in C

This folder contains C programs demonstrating the basic conditional statements used for decision-making.

## Programs Included

- if_statement.c
- if_else_statement.c
- elif_statement.c

## Concepts Covered

- if Statement
- if-else Statement
- else-if Ladder

## Language

- C

## How to Compile

```bash
gcc filename.c -o output
```

## How to Run

### Windows

```bash
output.exe
```

### Linux / macOS

```bash
./output
```
