# Addition Programs in C

This folder contains different types of addition programs implemented using functions in C language.

## Programs Included

1. addition.c
   - Function without arguments and without return value.

2. addition_with_args.c
   - Function with arguments and without return value.

3. addition_return.c
   - Function without arguments and with return value.

4. addition_with_args_return.c
   - Function with arguments and with return value.

## Language Used

- C Programming
