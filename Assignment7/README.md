# C Programming - Assignment 7

## 📖 Description

This repository contains C language solutions for Assignment 7. The programs focus on pattern printing, number sequences, loops, and arithmetic operations. These exercises help improve logic-building skills by using functions, iterative statements, and simple mathematical concepts.

## ✨ Features

- Function-Based Programs
- User Input Handling
- Pattern Printing
- Number Sequences
- Loops (for and while)
- Multiples of a Number
- Odd Number Generation
- Time Complexity Analysis

## 📂 Programs

| Question | Description |
|----------|-------------|
| Q1 | Print `$` and `*` pattern based on the given number |
| Q2 | Print numbers from 1 to N |
| Q3 | Print numbers from `-N` to `N` |
| Q4 | Print all odd numbers up to N |
| Q5 | Print the first five multiples of a given number |

## 🛠 Technology

- C

## ▶️ Running the Programs

Compile the program:

```bash
gcc filename.c -o output
```

Run:

```bash
./output
```

Windows:

```bash
output.exe
```

Example:

```bash
gcc Q1_Pattern.c -o Q1
Q1.exe
```
```
