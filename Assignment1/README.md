# Logic Building Assignment 01

## Description

This folder contains the solutions for **Logic Building Assignment 01** in **C Programming**. The programs are designed to improve logical thinking and strengthen the understanding of fundamental programming concepts.

## Objectives

* Develop logical thinking skills
* Practice C programming
* Understand functions, loops, and conditional statements
* Improve problem-solving techniques

## Programs Included

1. Divide Two Numbers
2. Print "Marvellous" 5 Times
3. Print Numbers from 5 to 1
4. Check Whether a Number is Divisible by 5
5. Print '*' Pattern

## Concepts Covered

* Functions
* Loops
* Conditional Statements
* User Input and Output
* Arithmetic Operations
* Pattern Printing

## Language

* C Programming

## Note

The complete problem statements are available in **Question.md**. Each C source file contains the solution for its corresponding question.
