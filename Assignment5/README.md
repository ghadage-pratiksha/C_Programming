# C Programming - Assignment 5

## 📖 Description

This repository contains C language solutions for Assignment 5.

The programs demonstrate the following concepts:

- Functions
- User Input
- Loops
- Conditional Statements
- Factors and Non-Factors
- Multiplication of Factors
- Summation
- Difference Calculation
- Time Complexity Analysis

## 📂 Programs

| Question | Program |
|----------|---------|
| Q1 | Calculate multiplication of all factors |
| Q2 | Display factors in decreasing order |
| Q3 | Display all non-factors of a number |
| Q4 | Calculate the sum of all non-factors |
| Q5 | Find the difference between the sum of factors and non-factors |

## 🛠 Language

- C

## ▶️ How to Run

Compile the program:

```bash
gcc filename.c -o output
```

Run the executable:

```bash
./output
```

or on Windows:

```bash
output.exe
```
